# Лабораторна робота №5 — Наука про дані: обмін результатами та початковий аналіз

Веб-додаток на базі **Streamlit** для інтерактивної фільтрації та візуалізації часових рядів VCI, TCI та VHI з даних лабораторної роботи №2.

---

## Функціонал

- **Dropdown** для вибору часового ряду: VCI, TCI або VHI
- **Dropdown** для вибору області України
- **Slider** для вибору інтервалу тижнів
- **Slider** для вибору діапазону років
- **Кнопка скидання** всіх фільтрів до початкових значень
- **Три вкладки**:
  - 📋 Таблиця — відфільтровані дані
  - 📈 Графік — часовий ряд за обраний діапазон
  - 🗺️ Порівняння — порівняння значень обраної області з іншими
- **Два checkbox** для сортування даних за зростанням / спаданням

## Встановлення та запуск

### 1. Клонування репозиторію

```bash
git clone <your-repo-url>
cd <repo-folder>
```

### 2. Створення та активація віртуального середовища

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 4. Запуск додатку

```bash
streamlit run app.py
```

Додаток відкриється у браузері за адресою `http://localhost:8501`.
