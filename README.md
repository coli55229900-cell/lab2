## Крок 1 — Створити віртуальне середовище (venv)

**Вікна:**

```
python -m venv venv
```

## Крок 2 — Активувати venv

**Вікна:**

```
venv\Scripts\activate
```

Після активації в терміналі з'явиться `(venv)` на початку рядка.

## Крок 3 — Встановити бібліотеки з вимогами.txt

```
pip install -r requirements.txt
```

## Крок 4 — Запустити програму

```
python lab4.py
```

Відкриється вікно з інтерактивним графіком гармоніки.

## Крок 5 — Взаємодія з інтерфейсом

- Переміщуй слайдери для зміни параметрів гармоніки та шуму
- Чекбокс **Show Noise** — показати/сховати шум на графіку
- Кнопка **Reset** — повернути початкові параметри
